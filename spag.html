Spaghetti Coder

Spaghetti Coder is a one night game project I made in late 2024. It is a game about Code and Spaghetti. The goal of the game is to avoid letting Spaghetti touch your RAM or CPU. I am not sure if this game is even possible to beat. Hell I am not even sure if it is fun, but it was a fun project to make. I made this game in Gamemaker Studio
